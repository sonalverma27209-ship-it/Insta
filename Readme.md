<h2> About</h2>

<p>
The most common form of online intelligence gathering is collecting
<b>publicly available information</b>. SHIVANSHI OSINT Tool is designed
to help researchers, students, and cybersecurity learners understand
how public data can be analyzed responsibly.
</p>

<p>
This tool <b>does not hack</b>, <b>does not bypass security</b>, and
<b>does not access private data</b>. It only works with information
that is already publicly accessible.
</p>

<h2> Quick Start and Setup for Kali Linux</h2>

<pre>
sudo apt update
sudo apt install git python3 curl wget -y
git clone https://github.com/sonalverma27209-ship-it/Insta.git
cd Insta
chmod +x *
python3 Insta.py -h
python3 Insta.py
</pre>

<hr>

<h2>Quick Start and Setup for Termux</h2>

<pre>
pkg update && pkg upgrade -y
pkg install git python curl wget -y
git clone https://github.com/sonalverma27209-ship-it/Insta.git
cd Insta
chmod +x *
python Insta.py
</pre>
